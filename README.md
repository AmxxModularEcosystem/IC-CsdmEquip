# [IC] CSDM Equip

Меню экопировки для CSDM режима.

## Команды

- `ic_csdm_equip_auto` - Выдача выбранных предметов или открытие меню, если предметы не были выбраны. Вызывается при каждом возрождении.
- `ic_csdm_equip_menu` - Открытие меню первой стадии выбора.
- `ic_csdm_equip_stage <stage>` - Открытие меню указанной стадии выбор.
- `ic_csdm_equip_stage_item <stage> <item>` - Получение указанного предмета из указанной стадии выбора.

Для создания команд для чата и подгонки под команды из других плагинов экипировки можно использовать [CommandAliases](https://github.com/AmxxModularEcosystem/CommandAliases).
